    Help

    What is the goal of the project?

    Nowdays, platforms have an extremely invasive privacy policy which relies on using user data in unethical ways. You give them a lot of data - ranging from ideas, music taste, content, political opinions, and much more than you think. By using codnap, you can freely watch and listen to content without the fear of prying eyes watching everything you are doing.

    Which Licence do codefoss use ?

    AGPL-3.0

    How codefoss runs ?

    Through your contributions and donations.

    How can I contribute to codefoss ?

    Whether you have ideas, translations, design changes, code cleaning, or real heavy code changes, help is always welcome. The more is done the better it gets! If you're interested in contributing, you can browse the issues or create a new one to discuss your feature idea. Every contribution is very welcome.
